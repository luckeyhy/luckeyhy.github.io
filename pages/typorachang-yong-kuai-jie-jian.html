Typora常用快捷键

Slug: https://luckeyhy.github.io/Typora

Date: 2022-6-16 18:15

Category: Record

Tags: blog, pelican

Author: luckeyhy

Summary: 本文记录一下Typora的常用快捷键

Typora常用快捷键：

菜单栏：

  1. 文件：alt + F
  2. 编辑：alt + E
  3. 段落：alt + P
  4. 格式：alt + O
  5. 视图：alt + V
  6. 主题：alt + T
  7. 帮助：alt + H

文件：

  1. 新建：ctrl + N
  2. 新建窗口：ctrl + shift + N
  3. 打开：ctrl + O
  4. 快速打开：ctrl + P
  5. 保存：ctrl + S
  6. 另存为：ctrl + shift + S
  7. 偏好：ctrl + ,
  8. 关闭：ctrl + W

编辑：

  1. 撤销：ctrl + Z
  2. 重做：ctrl + Y
  3. 剪切：ctrl + X
  4. 复制：ctrl + C
  5. 粘贴：ctrl + V
  6. 复制为MarkDown：ctrl + shift + C
  7. 粘贴为纯文本：ctrl + shift + V
  8. 全选：ctrl + A
  9. 选中当前行/句：ctrl + L
  10. 选中当前格式文本：ctrl + E
  11. 选中当前词：ctrl + D
  12. 跳转到文首：ctrl + Home
  13. 跳转到所选内容：ctrl + J
  14. 跳转到文末：ctrl + End
  15. 查找：ctrl + F
  16. 查找下一个：F3
  17. 查找上一个：shift + F3
  18. 替换：ctrl + H

段落：

  1. 标题：ctrl + 1/2/3/4/5/6
  2. 段落：ctrl + 0
  3. 增大标题级别：ctrl + =
  4. 减小标题级别：ctrl + -
  5. 表格：ctrl + T
  6. 代码块：ctrl + shift + K
  7. 公式块：ctrl + shift + M
  8. 引用：ctrl + shift + Q
  9. 有序列表：ctrl + shift + [
  10. 无序列表：ctrl + shift + ]
  11. 增加缩进：ctrl + ]
  12. 减少缩进：ctrl + [

格式：

  1. 加粗：ctrl + B
  2. 斜体：ctrl + I
  3. 下划线：ctrl + U
  4. 句中代码：ctrl + shift + `
  5. 删除线：alt + shift + 5
  6. 超链接：ctrl + K
  7. 图像：ctrl + shift + I
  8. 清除样式：ctrl +

视图：

  1. 显示隐藏侧边栏：ctrl + shift + L
  2. 大纲视图：ctrl + shift + 1
  3. 文档列表视图：ctrl + shift + 2
  4. 文件树视图：ctrl + shift + 3
  5. 源代码模式：ctrl + /
  6. 专注模式：F8
  7. 打字机模式：F9
  8. 切换全屏：F11
  9. 实际大小：ctrl + shift + 0
  10. 放大：ctrl + shift + =
  11. 缩小：ctrl + shift + -
  12. 应用內窗口切换：ctrl + Tab
  13. 打开DevTools：shift + F12
